A tiny description

This repository contains the research, planning, and documentation for the Dissonance project.

What goes here
- Research notes and veille
- Comparative studies and benchmarks
- Survey exports and summaries
- Personas and user research artifacts
- Design sketches, UI flows, and technical decisions
- Assets (images, audio, slides) and templates

How to add content
- Put materials in the matching folder (for example `research/`, `surveys/`, `personas/`, `design/`).
- For studies and reports, use a date-prefixed folder name: `YYYY-MM-DD_short-title`.
- Use the templates in `templates/` (for meeting notes, study writeups, issue templates).

Contributing
- Small edits: open a PR with a short description.
- Larger additions: open a draft PR and request a reviewer.
- Use the `CONTRIBUTING.md` for style and PR checklist.

Where to start
- See `planning/milestones.md` for milestone mapping.
- For benchmarks and comparisons, see `research/comparative-studies/` and `research/benchmarks/`.

(If you want a docs site published, I can scaffold MkDocs or Docusaurus and add a publish workflow.)
